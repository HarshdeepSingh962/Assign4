# Tamagotchi Game

## Description
Tamagotchi Game is a mobile application developed using React Native. It simulates the experience of taking care of a virtual pet, where users can interact with the pet by feeding, cleaning, resting, and playing with it.

## Dependencies
- React
- React Native
- Expo
- expo-av

## File Paths
- `TamagotchiGame.js`: Contains the main logic and UI components for the Tamagotchi game.
- `assets/`: Directory containing image and sound assets used in the game.

## Connectivity Requirements
This application does not have any specific connectivity requirements. It can run offline without needing an internet connection.

## Usage
1. Clone the repository.
2. Install dependencies using `npm install`.
3. Run the application using Expo by running `npx expo start`and enjoy the app.
